Column guide

  • Card name - collector title (Series 1 from EmpireZ TCG.xlsx where available; Series 2 from legacy config display name).
  • Classname - DayZ / Pokemon mod item classname used when the card spawns.
  • Legacy display - raw displayName from live config.cpp before xlsx override.
  • Collector rarity - Series 1 spreadsheet rarity; Series 2 inferred from texture filename (_rare_, Super Rare).
  • Mod pack rarity - highest tier that classname appears in across Basic/Jungle/Fossil pack loot tables (Energy, Trainer, Common through Legendary).
  • EmpireZ box slot - attachment slot on EmpireZ_CardBox_Series1/2 in config_new.cpp.
  • Legacy box slot - original Pokemon collector box slot (Pokemon_NNN, trainer, energy) if using vanilla boxes.
  • Notes - for cards not in pack loot: why (Team Rocket commented, variant not listed, shared slot sibling drops instead, missing from pack scripts, etc.).

Notes categories (not in packs)

  • Team Rocket - commented, unfinished - //TeamRocket lines in seal-box script; rainbow energy / trainers 25-31; Rocket pack never implemented.
  • Variant not in packs - base classname (e.g. Pokemoncard004) is in loot tables but this suffix variant (R, H, RH) is not.
  • Set variant not in packs - Jungle (J) or Fossil (F/FH) print exists in packs under a related classname, not this one.
  • Shared legacy slot - another print for the same Pokemon_* / trainer slot drops from packs.
  • Collector vs pack mismatch - EmpireZ checklist number maps to a different legacy slot than the classname packs actually drop.
  • Missing from all pack scripts - classname never Insert()'d; card still defined in config.
